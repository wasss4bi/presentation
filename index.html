Таблица принятия решений

Поздняков Дмитрий, 41 ИСР

Что это?

Таблица принятия решений - инструмент тестирования, который объединяет условия в удобочитаемую таблицу, позволяющую быстро выявлять баги

Когда используется?

Таблица принятия решений используется, когда необходимо провести немало тестов, при этом сделать это быстро и без надобности удерживать в голове всех тонкостей тестирования

Пример

Проверка человека при взятии на работу

Тест 1 Тест 2 Тест 3 Тест 4
Условия
Стаж работы 3 года Да Да Нет Нет
Есть семья Да Нет Да Нет
Результат
Взят на работу? Да Да Да Нет

Пример 2

Вычисление скидки для покупателя при покупке товаров в магазине

Тест 1 Тест 2 Тест ... Тест n
Условия
Потратил 100 200 ... n
Результат
Скидка 2% 4% ... <30%

Алгоритм применения

  • В первый столбец выписать все входные данные для тестирования
  • Написать действие в последних строках, описывающее происходящее с входными данными
  • Для каждого тест-кейса создаём свой столбец и заполняем входные данные

Плюсы

  • Наглядность - удобнее таблицы ещё не придумали способа структурирования информации
  • Удобство отслеживания багов
  • Все тест-кейсы видны практически на одном экране

Минусы

  • Может быть избыточным решением для слишком маленького проекта
  • Если будет очень много вводных данных, таблица получится очень большой
