Хуки отрисовки

useState()


						import React, { useState } from 'react';
						function Component(){
							const [message, setMessage] = useState("bambam"); // по умолчанию
							console.log(message) // "bambam";
							
							setMessage("tutu");
							console.log(message) // "tutu";
						}

При изменении состояния обновляется компонент и перерисовывается заново

useEffect()


						import React, { useState, useEffect } from 'react';
						function Component(){
							const [boolValue, setBoolValue] = useState(false);
							useEffect(()=>{
								console.log("bebe")
								if(boolValue){
									console.log("it's true!")
								}
							}, [boolValue]) // Зависимости

							setBoolValue(!boolValue); // После этой строки хук сработает ещё раз
						}

useEffect отрабатывает при первом рендере компонента, а при изменении элемента в массиве зависимостей (boolValue) перерисовывает компонент с новыми значениями

useContext()

useContext позволяет напрямую обратиться к необходимой переменной из родительского в дочернем элементе

useMemo()

Используется для мемоизации значений. То есть, исключает повторное вычисление того, что уже было посчитано

Suspense, lazy

Suspence и lazy используются подгрузки компонентов только по необходимости. Используется для оптимизации приложения
