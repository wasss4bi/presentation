LSP

LSP - Language Server Protocol - это протокол, помогающий IDE общаться с разными языками программирования с помощью сервера

TS LS

TS LS - TypeScript Language Server - Один из серверов, к которому обращается IDE через LSP. Конкретно этот обеспечивает для IDE комфортную работу с TypeScript и JavaScript

CSS LS

CSS LS - Cascading Style Sheet Language Server - Также один из серверов, к которому обращается IDE через LSP. Он обеспечивает для IDE комфортную работу с CSS

Алгоритм

  • IDE хочет помочь найти ошибки в синтаксисе языка и отправляет запрос на сервер
  • Запрос летит по LSP и достигает нужного сервера
  • Сервер указывает на ошибки и обратно по LSP отправляет их на сервер
  • В IDE подсвечиваются ошибки

Схематично

						
									 					TypeScript Language Server - JS и TS
														Python Language Server - Python
						IDE <-> LSP <->	Intelephense - PHP
														Java Language Server - Java
														clangd - C и C++
Существует множество разных LSP серверов

Emmet

Emmet - встроенный в vs code инструмент для быстрого написания HTML и CSS

HTML 

						
							! - создаст базовую структуру html
							h1{heading} - создаст заголовок h1 со словом heading
							p>lorem10 - создаст тег p с 10 словами из lorem ipsum
							div.block - создаст div с классом block
							div#block - создаст div с id block
							div>p - создаст div с тегом p внутри себя
							ul>li*5 - создаст блок списка с пятью элементами

Такой подход позволяет очень сильно ускорить процесс вёрстки.
